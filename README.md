# 简介
微人事是一个前后端分离的人力资源管理系统，项目采用SpringBoot+Vue开发。

## 安装依赖
```
npm install
```

### 启动
```
npm run serve
```

### 打包
```
npm run build
```

### 目录结构

```html
├── public  打包所需静态资源
└── src
    ├── utils  工具函数
    ├── components  业务组件
    ├── router  路由配置
    ├── store  Vuex配置
    ├── view  页面文件
    └── assets  项目静态资源
```
